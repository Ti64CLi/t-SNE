# t-SNE Implementation in Python

This is my own implementation of the data visualization algorithm t-SNE.

I used [the original paper by Laurens van der Maaten and Geoffrey Hinton](https://lvdmaaten.github.io/publications/papers/JMLR_2008.pdf) for my implementation, especially the early exaggeration and adaptative momentum parts.
